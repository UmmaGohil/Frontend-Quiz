# ⁉️Front-end Quiz

A small quiz app built with create-react-app.

### 👩‍💻Technical Details

- CSS styling used throughout the application

- Hooks implemented to manage state

- Dummy data via json file

- React testing-library used for unit tests

### 🔧How to Run the App

- `yarn`

- `yarn start`
